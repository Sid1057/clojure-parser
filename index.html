clojure-parser

Library to produce an informative data from some text.

Supported data types:

  • Links
  • Dates
  • Times
  • Emails
  • Phone-numbers
  • Proper names

Commands:

Supported commands:
  • Check links
  • Wikipedia Proper name
In the pipeline:
  • Send-mail message to email
  • Remind message time and maybe date
  • Phone-country phone number

More about commands

Usage(just a simple example of demo version):

input.txt

Text for parsing

Console:
  lein deps
  lein compile
  lein repl
REPL:
  (main)

input.txt:
